# Attendance Bot

A Python-based attendance management system designed to automate attendance tracking and provide admin/student access controls.

## Project Overview

This project implements an attendance bot system that allows administrators to manage student records and enables students to access their attendance information securely. The system is built with Python and includes database management, user authentication, and role-based access controls.

## Features

- **Admin Management** - Administrative functions for managing attendance records
- **Student Access** - Secure student portal for viewing attendance information
- **Database Integration** - SQL-based data persistence and management
- **User Authentication** - Role-based access control for admins and students

## Project Structure
